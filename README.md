# Washing Machine 
## Introduction
This project will imitate the behaviour of a washimg machine using modelsim.

![](images/FSM1.png)

### Sensors
- full (at the top)
- empty (at the bottom)

### Outputs
- Inlet valve
- Outlet valve
- Alarm
- Motor forward
- Motor Reverse
- Lock door

### Timers
- washing
- Rinsing 
- Spinning
- alarm
